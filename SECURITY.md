# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via:

1. **Email**: [yudhistiramt01@gmail.com](mailto:yudhistiramt01@gmail.com)
2. **GitHub Security Advisories**: Use the [Security Advisories](https://github.com/arthiccc/ymt/security/advisories) page

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

We aim to respond within 48 hours and will work with you on a coordinated disclosure.
